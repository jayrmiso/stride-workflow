# Stride Kit

Use this skill for UI work that should match or evolve from an existing reference.

## Use when

- The user shares a screenshot or design reference.
- The user wants consistent UI rather than a fresh one-off pattern.
- The task is about components, screens, or layout behavior.

## Do

- Study the reference carefully.
- Reuse the visual language already present.
- Keep the UI coherent across repeated surfaces.
