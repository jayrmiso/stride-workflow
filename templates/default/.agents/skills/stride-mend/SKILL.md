# Stride Mend

Use this skill when fixing issues discovered during review or manual testing.

## Use when

- A bug was found.
- A review requested changes.
- The implementation needs a focused repair.

## Do

- Fix the issue without broadening scope.
- Keep the frame intact.
- Re-check the result after the fix.
