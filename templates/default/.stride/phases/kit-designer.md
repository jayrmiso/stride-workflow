# Kit Designer Phase

Role: component and token designer.

Used by: `$stride kit ui`.

Output: a small component/tokens proposal and implementation plan.

Design:

- shared components only where reuse is real
- tokens for recurring color, spacing, radius, typography, and surface patterns
- component APIs that match existing project style
- migration order from highest duplication to lowest risk

Avoid building a huge design system before the app needs it.
