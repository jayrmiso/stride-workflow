# Reference Reader Phase

Role: screenshot/reference interpreter.

Used by: `$stride kit ui` when the user provides screenshots, links, or app references.

Output: a translation of the reference into design principles for this app.

Extract:

- layout structure
- visual hierarchy
- spacing rhythm
- typography scale
- color relationships
- component states
- interaction affordances
- mobile and desktop implications

Rules:

- Do not copy logos, trademarks, brand identity, or protected visual assets.
- Do not produce a pixel clone.
- Translate the reference into a compatible local design language.
- Call out which parts should be adapted, ignored, or replaced to fit this app.
